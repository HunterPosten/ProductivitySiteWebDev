How I made this site:

In the process of making this site I had to use many tools. the base structure of each page is coded in HTML this allowed me to construct and modify each page structurally without worrying about how the styling would be affected. When I did want to concern myself with styling I would modify the site wide CSS file. having each page in the site use the same css file allows this site to have a cohesive look. Finally since there are a few elements on my site that are interactive i needed to include some javascript. I wrote all of this using Sublime text and tested each page on Opera and Chrome. Overall this project has definitely forced me to learn a lot about css and javascript since i only new the bare bones basics before.
